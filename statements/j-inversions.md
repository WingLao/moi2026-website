## 題目說明

給定一個長度為 N 的整數序列 A，若 i < j 且 A[i] > A[j]，則稱 (i, j)
為一個逆序對。請計算序列中逆序對的總數量。

## 輸入格式

第一行包含一個正整數 N（1 ≤ N ≤ 10^5）。
第二行包含 N 個整數（1 ≤ A[i] ≤ 10^9），以空格分隔。

## 輸出格式

一行，輸出逆序對的數量。

## 範例輸入

```text
5
3 1 2 5 4
```

## 範例輸出

```text
3
```

## 說明

逆序對為 (3,1), (3,2), (5,4)，共 3 對。
