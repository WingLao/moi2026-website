# 病毒擴散 (spread)

> 此網站版本改用 Markdown 題面，範例輸入 / 輸出直接對齊 `data` 內的 `*-0.in` / `*-0.out`，避免題面與測資不一致。

|項目|內容|
|---|---|
|任務名稱|`spread`|
|組別|`J`|
|測資目錄|`data/J/spread`|
|原始 PDF|`J-spread.pdf`|
|對齊範例|`spread-0.in` / `spread-0.out`|

## 題目說明

給定一個 N×M 的網格，每個格子初始狀態為 0（健康）或 1（已感染）。每分鐘，每個已感染格子
會向上下左右四個方向的健康格子傳播病毒。求多少分鐘後所有格子都被感染？若永遠無法全部感染
（存在無法到達的格子），輸出 -1。

## 輸入格式

第一行包含兩個正整數 N、M（1 ≤ N, M ≤ 50）。
接下來 N 行，每行 M 個整數（0 或 1），以空格分隔。

## 輸出格式

一行，輸出最少需要的分鐘數，若無法全部感染則輸出 -1。

## 範例輸入

```text
3 3
0 1 0
0 0 0
0 0 0
```

## 範例輸出

```text
3
```

## 說明

第0分鐘：(1,1)感染。第1分鐘：(0,1),(1,0),(1,2)。第2分鐘：擴散。第3分鐘：全部感染。
