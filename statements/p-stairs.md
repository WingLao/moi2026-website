## 題目說明

有一段樓梯共 N 級，每次可以走 1 步或 2 步，請計算從地面（第 0 級）走到第 N
級共有多少種不同的走法。

## 輸入格式

一行，包含一個正整數 N（1 ≤ N ≤ 30）。

## 輸出格式

一行，輸出走到第 N 級的方法數。

## 範例輸入

```text
1
```

## 範例輸出

```text
1
```

## 說明

走到第5級的方法：11111、1112、1121、1211、2111、122、212、221，共8種。
