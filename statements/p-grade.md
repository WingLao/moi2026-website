# 成績排序 (grade)

> 此網站版本改用 Markdown 題面，範例輸入 / 輸出直接對齊 `data` 內的 `*-0.in` / `*-0.out`，避免題面與測資不一致。

|項目|內容|
|---|---|
|任務名稱|`grade`|
|組別|`P`|
|測資目錄|`data/P/grade`|
|原始 PDF|`P-grade.pdf`|
|對齊範例|`grade-0.in` / `grade-0.out`|

## 題目說明

給定 N
個學生的姓名和分數，請按分數由高到低排序後輸出。若分數相同，按姓名字典序由小到大排序。

## 輸入格式

第一行包含一個正整數 N（1 ≤ N ≤ 20）。
接下來 N 行，每行包含一個不含空格的姓名（長度 ≤ 10）和一個整數分數（0 ≤ score ≤
100），以空格分隔。

## 輸出格式

N 行，按排序後的順序每行輸出「姓名 分數」。

## 範例輸入

```text
3
Alice 90
Bob 85
Carol 90
```

## 範例輸出

```text
Alice 90
Carol 90
Bob 85
```

## 說明

Alice 和 Carol 同分90，依字典序 Alice < Carol 排在前面。
